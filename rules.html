Game Rules

Objective

Your goal is to slide numbered tiles on a grid to combine them to create a tile with the number 2048.

Starting the Game

The game begins with two tiles already in the grid, having a value of either 2 or 4, and another such tile appears in a random empty space after each turn.

Gameplay

  1. Tiles slide as far as possible in the chosen direction until they are stopped by either another tile or the edge of the grid.
  2. If two tiles of the same number collide while moving, they will merge into a tile with the total value of the two tiles that collided.
  3. The resulting tile cannot merge with another tile again in the same move. Higher-scoring tiles emit a soft glow.

Winning the Game

The game is won when a tile with a value of 2048 appears on the board.

Additional Rules

  • If a move causes three consecutive tiles of the same value to slide together, only the two tiles farthest along the direction of motion will combine.
  • If all four spaces in a row or column are filled with tiles of the same value, a move parallel to that row/column will combine the first two and last two.
  • A scoreboard on the upper-right keeps track of the user's score. The user's score starts at zero, and is increased whenever two tiles combine, by the value of the new tile.
